# Default Images for Marketing Website

This folder contains default/fallback images for the marketing website. These images are used when no custom images are provided through the CMS.

## Image Sources and Recommendations

For high-quality, free business images, consider these sources:

### Recommended Free Stock Photo Sites:
1. **Unsplash** (https://unsplash.com) - High-quality, free photos
2. **Pexels** (https://pexels.com) - Free stock photos and videos
3. **Pixabay** (https://pixabay.com) - Free images, vectors, and videos

### Recommended Business Image Categories:
- Professional office environments
- Team collaboration and meetings
- Modern technology and devices
- Clean, minimalist business settings
- Handshakes and business interactions
- Modern architecture and buildings

### Image Specifications:
- **Hero Background**: 1920x1080px (16:9 ratio)
- **Service Images**: 400x300px (4:3 ratio)
- **About Images**: 600x400px (3:2 ratio)
- **Logo Placeholder**: 200x60px (flexible)

### Current Placeholder Images:
- `hero-bg.jpg` - Hero section background
- `service-1.jpg` - Service placeholder 1
- `service-2.jpg` - Service placeholder 2  
- `service-3.jpg` - Service placeholder 3
- `about-team.jpg` - About page team photo
- `logo-placeholder.png` - Logo placeholder

## Usage in Components

These images are referenced in the fallback content of components when CMS images are not available.

## Replacing Images

To replace these placeholder images:
1. Download high-quality images from the recommended sources
2. Resize them to the specified dimensions
3. Replace the files in this folder with the same names
4. Ensure images are optimized for web (compressed but high quality)

## License

Make sure any images you use are properly licensed for commercial use. The recommended sources above provide images that are free for commercial use, but always check the specific license for each image.
