## Quick Download Commands

If you have `curl` or `wget`, you can download placeholder images:

```bash

cd public/images

# Download sample images (these are just examples - replace with actual URLs)
curl -o hero-bg.jpg "https://images.unsplash.com/photo-1497366216548-37526070297c?w=1920&h=1080&fit=crop"
curl -o service-1.jpg "https://images.unsplash.com/photo-1552664730-d307ca884978?w=400&h=300&fit=crop"
```

## After Downloading

1. Replace the SVG files with the downloaded JPG files
2. Update the file extensions in your components if needed
3. Optimize the images for web (compress them)
4. Test that all images load correctly

## Image Optimization Tools

- **TinyPNG**: https://tinypng.com/ (online compression)
- **ImageOptim**: https://imageoptim.com/ (Mac app)
- **Squoosh**: https://squoosh.app/ (Google's web app)

Remember to always check the license of any images you download and ensure they're free for commercial use!
